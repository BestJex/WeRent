# travel
## Project Background
该项目爬取了链家,58找房等房源数据,基于数据过滤引擎,采用用户协同过滤算法为户提供个性化智能租房房源信息。前端采用了Vue-cli+ elmentui框架搭建了主页、登录页、详情页等界面，主要使用了router、自定义组件、axios发送异步的请求。网站设计参考国外设计网站，简约清晰。
